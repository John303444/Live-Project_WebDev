# Live-Project_WebDev
Role: Developer
Tech Stack: HTML, CSS, Javascript

  A clean and modern web application dedicated to Judo, built using HTML, CSS, and JavaScript. The project focuses on elegant simplicity, responsive design, and a smooth user experience with a night theme aesthetic.

  Features:Responsive Navigation Bar:
A fixed, easy-to-use navbar that helps users quickly navigate different sections of the site.

Night Theme Design:
A dark, minimal, and modern look that reduces eye strain and enhances visual focus.

Interactive Form:
Includes a simple form that allows users to submit their details or feedback.
